\section{任务问题描述}

本部分负责图像噪声处理的基础任务，具体包括：

\begin{enumerate}
\item 空域滤波：对含高斯噪声和椒盐噪声的图像，分别实现均值滤波和中值滤波，观察并对比去噪效果。
\item 频域分析：利用离散傅里叶变换（DFT）将含噪图像变换到频域，分析不同噪声在频域的分布特性。
\item 频域滤波：根据频域分析结果，设计合适的频域滤波器进行去噪，并与空域方法进行对比。
\item 结果对比：对实验结果进行定性分析，综合评估各滤波方法的性能。
\end{enumerate}

数据集：使用不含噪声的原始图像，通过代码分别添加高斯噪声和椒盐噪声，生成含噪图像用于后续处理。噪声参数设置如下：
\begin{itemize}
\item 高斯噪声：均值为0，标准差为25（像素值范围0-255）
\item 椒盐噪声：噪声密度为2\%（盐噪声和椒噪声各占1\%）
\end{itemize}

\section{原理分析}

\subsection{图像噪声模型}

\subsubsection{高斯噪声}

高斯噪声的概率密度函数服从正态分布，在空间域表现为每个像素独立叠加随机值：

\begin{equation}
p(z) = \frac{1}{\sqrt{2\pi}\sigma}e^{-(z-\mu)^2/(2\sigma^2)}
\end{equation}

其中$\mu$为均值，$\sigma$为标准差。高斯噪声属于加性噪声，在频域中功率谱密度近似常数，呈现全频段均匀分布特性。

\subsubsection{椒盐噪声}

椒盐噪声表现为图像中随机出现的黑白像素点（灰度值为0或255），其概率分布为：

\begin{equation}
p(z) = 
\begin{cases} 
P_a, & z = a \\
P_b, & z = b \\
0, & \text{其他}
\end{cases}
\end{equation}

椒盐噪声在空间域为孤立脉冲点，在频域等价于高频冲击信号，呈现高频成分为主的分布特性。

\subsection{空域滤波原理}

\subsubsection{均值滤波}

均值滤波用邻域像素的平均值替代中心像素值：

\begin{equation}
g(x,y) = \frac{1}{mn}\sum_{(s,t)\in S_{xy}} f(s,t)
\end{equation}

其中$S_{xy}$表示中心在$(x,y)$、大小为$m\times n$的矩形邻域。均值滤波本质是低通滤波，对高斯噪声有较好的平滑效果，但会使边缘和细节模糊。

\subsubsection{中值滤波}

中值滤波用邻域像素的中值替代中心像素值：

\begin{equation}
g(x,y) = \underset{(s,t)\in S_{xy}}{\text{median}}\{f(s,t)\}
\end{equation}

中值滤波属于非线性滤波，对椒盐噪声效果显著，能较好保留边缘。

\subsection{频域分析原理}

图像的二维离散傅里叶变换（DFT）定义为：

\begin{equation}
F(u,v) = \sum_{x=0}^{M-1}\sum_{y=0}^{N-1} f(x,y)e^{-j2\pi(ux/M+vy/N)}
\end{equation}

其中$M\times N$为图像尺寸。幅度谱$|F(u,v)|$反映各频率分量的能量分布，相位谱$\angle F(u,v)$反映相位信息。通过分析噪声图像与原图的频谱差异，可以揭示噪声的频域分布规律。

\subsection{频域滤波原理}

在频域分析的基础上，根据噪声的频域分布特性，分别设计针对性的频域滤波器。

\subsubsection{维纳滤波（针对高斯噪声）}

频域分析表明，高斯噪声在全频段均匀分布。针对这一特性，采用维纳滤波进行自适应处理：

\begin{equation}
F'(x,y) = \mu + \frac{\max(\sigma^2-n^2,0)}{\max(\sigma^2,n^2)}[f(x,y)-\mu]
\end{equation}

其中$\mu$为局部均值，$\sigma^2$为局部方差，$n^2$为噪声方差。维纳滤波能根据局部统计特性自适应调整滤波强度：在平坦区域（$\sigma^2 \approx n^2$）实现强滤波，在边缘区域（$\sigma^2 \gg n^2$）保留原始像素值。

\subsubsection{巴特沃斯低通滤波（针对椒盐噪声）}

频域分析表明，椒盐噪声主要分布在高频区域。针对这一特性，采用巴特沃斯低通滤波器抑制高频成分：

\begin{equation}
H(u,v) = \frac{1}{1+[D(u,v)/D_0]^{2n}}
\end{equation}

其中$D(u,v)=\sqrt{u^2+v^2}$为到频率中心的距离，$D_0$为截止频率，$n$为阶数。巴特沃斯滤波器具有平滑的过渡带，能有效避免理想滤波器产生的振铃效应。

\section{算法设计与分析}

\subsection{整体流程}

\begin{enumerate}
\item 读取原始图像，分别添加高斯噪声和椒盐噪声
\item 空域滤波：应用均值滤波和中值滤波（滤波核大小为$3\times3$）
\item 频域分析：对原图和噪声图进行DFT变换，计算幅度谱和相位谱，对比频谱差异
\item 频域滤波器选择：根据频域分析结果——
\begin{itemize}
\item 高斯噪声全频段均匀分布 $\rightarrow$ 选择维纳滤波
\item 椒盐噪声高频成分为主 $\rightarrow$ 选择巴特沃斯低通滤波
\end{itemize}
\item 频域滤波：应用选定的滤波器进行去噪处理
\item 结果评估：生成对比图，进行定性分析
\end{enumerate}

\subsection{关键参数选择}

\begin{itemize}
\item 滤波核大小：$3\times3$，在去噪能力和细节保留之间取得平衡，$3\times3$核能更好地保留图像细节
\item 维纳滤波噪声方差估计：$n^2=500$。由于高斯噪声的标准差$\sigma=25$，理论上噪声方差为$\sigma^2=625$。实际使用$n^2=500$，略低于理论值，使滤波器的去噪强度略微降低，从而更好地保留图像细节，避免过度平滑
\item 巴特沃斯低通截止频率$D_0=30$：根据频域分析结果，椒盐噪声主要集中在较高频段，选择较小的截止频率以有效滤除高频噪声分量
\item 巴特沃斯阶数$n=1$：一阶巴特沃斯滤波器具有最平滑的过渡带，能最大程度避免振铃效应，保留更多中频细节信息
\end{itemize}

从图中可以观察到：

\begin{itemize}

\item 均值滤波对高斯噪声的平滑效果较为明显，但图像整体变得模糊，边缘和纹理细节有所损失

\item 中值滤波处理高斯噪声时，去噪效果不如均值滤波彻底，图像仍保留部分颗粒感
\end{itemize}

\item 针对椒盐噪声（高频为主），巴特沃斯低通滤波通过抑制高频分量有效去除噪声，一阶平滑设计避免了振铃效应

\subsection{空域与频域滤波效果综合对比}


从综合对比可以看出：

\begin{itemize}
\item 针对\textbf{高斯噪声}：维纳滤波（频域自适应方法）效果优于均值滤波（空域方法）。维纳滤波在平坦区域实现强滤波以去除噪声，在边缘区域弱滤波以保留细节，这正是其自适应特性的优势体现
\item 针对\textbf{椒盐噪声}：中值滤波（空域非线性方法）效果优于巴特沃斯低通滤波（频域方法）。中值滤波能够精确去除孤立的椒盐像素点，且不会造成图像整体模糊；而低通滤波在去除高频噪声的同时，不可避免地损失了部分高频细节信息
\item 综合结论：高斯噪声的最佳处理方法为维纳滤波，椒盐噪声的最佳处理方法为中值滤波。这两种方法的共同特点是能够根据图像局部特征进行自适应或选择性处理，而非简单地对整幅图像施加统一的滤波操作
\end{itemize}

\section{总结}

本基础任务完成了以下工作：

\begin{enumerate}
\item 空域滤波实现：使用$3\times3$滤波核实现了均值滤波和中值滤波。均值滤波对高斯噪声效果较好，中值滤波对椒盐噪声效果显著，能够几乎完全消除噪声点同时保持边缘清晰。
\item 频域特性分析：通过DFT变换和频谱可视化，系统分析了高斯噪声和椒盐噪声的频域分布规律。幅度谱分析、频率剖面分析和频谱差异分析一致表明：高斯噪声全频段均匀分布，椒盐噪声高频成分为主。
\item 频域滤波器选择与设计：基于频域分析结果，为高斯噪声选择维纳滤波（自适应全频段处理），为椒盐噪声选择巴特沃斯低通滤波（截止频率$D_0=30$，阶数$n=1$）。滤波器的选择完全基于噪声频域特性分析，而非简单枚举尝试。
\item 结果分析与对比：通过多组对比图的定性分析，验证了基于频域特征选择滤波器的有效性。综合对比表明，高斯噪声最佳处理方法为维纳滤波，椒盐噪声最佳处理方法为中值滤波。频域分析方法能够揭示噪声的本质特征，为滤波器选择提供理论指导，避免盲目尝试，体现了信号与系统中频域分析思想在图像处理中的重要应用价值。
\end{enumerate}
